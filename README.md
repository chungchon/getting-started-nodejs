# Getting started with nodejs
## Installation
### Windows:
1. download msi installer from [Nodejs website](https://nodejs.org/en/)
2. execute the msi installer, follow the instruction to install nodejs
3. restart your computer

**Test the environment**  
To make sure you've installed nodejs successfully, please run the following command in PowerShell:
```
node -v
npm -v
```
The above command will show you the version of installed nodejs and node package manager.  
Visit http://blog.teamtreehouse.com/install-node-js-npm-windows for guide with pictures

## Install packages in NPM
After installation, you can run npm to install the package with the sample code:
```
npm install <module_name>
```
